perlSGML

dtd.pl, dtd2html, dtddiff, dtdtree, sgml.pl, stripsgml

perlSGML is a collection of Perl programs and libraries for processing SGML documents. The following software is available in the perlSGML distribution:

  • dtd.pl 2.2.0
  • dtd2html 1.4.0
  • dtddiff 1.1.0
  • dtdtree 1.2.0
  • sgml.pl 0.1.0
  • stripsgml 0.1.1

Availability
